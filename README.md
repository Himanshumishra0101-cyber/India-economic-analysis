# India-economic-analysis
Economic analysis dashboard using Python and World Bank API data

## Features

* GDP trend analysis
* GDP growth visualization
* GDP per capita (PPP) analysis
* Population analysis
* Sector-wise economic analysis:

  * Agriculture
  * Industry
  * Services
* Correlation analysis between GDP growth and sector contribution

## Technologies Used

* Python
* pandas
* matplotlib
* requests
* numpy

## Data Source

World Bank API

## Objective

The goal of this project is to analyze India's macroeconomic trends and explore relationships between sectoral composition and GDP growth using real-world economic data.

## Key Learning Outcomes

* API handling in Python
* Data cleaning and transformation
* Economic data visualization
* Correlation analysis
* Analytical thinking using macroeconomic indicators
